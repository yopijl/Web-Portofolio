Telkomsel JPP Planning Dashboard

The objective of this project was to create an interactive and informative dashboard using Looker Studio to visualize Telkomsel's data for the first and second semesters. This dashboard allows stakeholders to easily analyze and interpret budget and performance data, enabling more informed decision-making.

Sentiment Analysis of Customer Reviews

In this project, I developed a sentiment analysis system for Amazon customer reviews using Python, comparing NLTK's VADER and Huggingface's Roberta Transformers. The aim was to classify review sentiments and evaluate the performance of these two approaches. This comparison helps in understanding the strengths and weaknesses of each method for more accurate sentiment analysis.

Car Price Prediction

The "Car Price Prediction" project is a regression problem targeting the "PRICE" variable. Using RMSE and R2 as evaluation metrics, the model achieves RMSE 1812.2707 and R2 0.9584, indicating a high level of accuracy in predicting car prices.

Website Perwira Sekti

Website-Perwira Sekti adalah aplikasi web yang menyediakan informasi perizinan pangan di Kota Semarang. Menampilkan Dashboard PIRT dan PKP, serta panduan alur dan persyaratan perizinan PIRT dan SLHS, proyek ini bertujuan mempermudah akses informasi perizinan pangan dan meningkatkan transparansi serta efisiensi proses perizinan.

Sistem Deteksi Stunting

Sistem prediksi stunting adalah aplikasi web yang menggunakan data antropometri anak dan Machine Learning untuk mendeteksi status stunting dan memberikan saran penanganan. Tujuannya adalah membantu orang tua dan tenaga medis dalam mendeteksi masalah stunting secara dini dan mengambil langkah-langkah yang sesuai.

Sign Language Detection

Developed a BISINDO sign language detection system using YOLOv5-NAS-S to aid communication for individuals with hearing and speech impairments, aligning with SDGs 4, 8, and 10. Utilized transfer learning, Super-Gradients, and Roboflow for data augmentation, achieving a model with 97.2% mAP and 99.6% Recall. Future work includes real-time integration and broader practical applications.

Visignify Application

Visignify is a unique, user-centric platform that offers a transformative solution by revolutionizing how individuals with disabilities communicate, access information, and engage with the world. This app includes features such as object, sign detection using machine learning, OCR for text-to-speech functionality, voice output for location, battery, weather information, learn feature, and a "Deaf Note" feature.
